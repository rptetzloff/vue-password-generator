# vue-password-generator

A simple password generator written in Vue with Vuetify.

## Dependencies

* Vue >= 2
* Vuetify >= 1.5
* vue-clipboard2 >= 0.3.0

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
