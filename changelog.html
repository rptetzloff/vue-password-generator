v2.0.0
2025
Major
Full rewrite — six generator modes
Complete rebuild with Vue 3 Composition API, Vite, and six distinct password generation modes replacing the single-mode v1 generator.
Added — New generator modes
  • Advanced mode — per-character-type minimum and maximum count sliders, plus an interactive symbol chip picker with All / Common / None presets.
  • Words mode — dictionary-based passwords from a curated word list; configurable word count (2–20), separator, capitalization, prefix/suffix, and leet-speak.
  • Passphrase mode — slot-based builder with adjective, adverb, noun, and verb slots; each slot has an independent thematic category (Animals, Colors, Mood, etc.).
  • Mad Lib mode — 12 narrative sentence templates; each typed placeholder gets its own category selector; readable phrase preview shown alongside the generated password.
  • Numbers mode — numeric PIN generator with configurable digit count and limits on consecutive repeated and sequential digits.
Added — Shared features
  • Separator picker — 14 built-in options (None, Space, Hyphen, Underscore, Period, Dollar, random symbols/numbers, combinations) plus a free-text Custom option.
  • Prefix & Suffix — independent affix pickers with the same options as the separator, plus two suffix-only mirror modes (Mirror Prefix and Mirror Prefix with new digits).
  • Capitalization modes — Title Case, lowercase, UPPERCASE, random-per-letter, alternating-per-word, and random-per-word.
  • Leet speak substitutions — 10 toggleable character substitutions (a→@, e→3, i→1, o→0, s→$, t→+, l→!, b→8, g→9, z→2) with All / None presets.
  • Word pills — each word in the output appears as a clickable pill that swaps only that word without regenerating the whole password.
  • Affix lock — lock icon freezes the current prefix, separator, and suffix while still allowing individual word re-rolls.
  • Generation history — per-tab history strip showing recent passwords; clicking any entry restores it to the output.
  • History size control — global header chip (Off / 5 / 10 / 20 / 50) configures the history limit across all tabs.
  • Settings persistence — all options saved to localStorage and restored on next visit.
Improved
  • Simple mode retained and refined — character set checkboxes, length slider (6–128), copy button with animated confirmation.
  • Responsive layout — tabs, cards, and affix pickers adapt cleanly from mobile to wide desktop.
  • Privacy notice in the header making the local-only guarantee explicit to users.
Architecture
  • Migrated from Vue 2 (Options API) to Vue 3 Composition API.
  • Build tooling switched from Vue CLI to Vite 5.
  • Vue loaded as an ES module from CDN — no framework bundling.
  • Word data split into wordlist.txt (Words mode) and words.json (Passphrase / Mad Lib) for efficient per-mode loading.
  • Shared composables (useHistory, useCopyPassword, useNotification, persistedRef) eliminate cross-component duplication.
